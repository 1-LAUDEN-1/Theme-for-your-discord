Guide

1. Install Better Discord - https://betterdiscord.app/

2. Download themes "Default-icons.theme.css" and "Small-icons.theme.css"

3. Click settings your discord - Themes - Click "Open Themes Folder"

4. Copy themes from your downloads to folder Themes

5. Turn on theme

Install plugins

1. Open folder "plugins" - copy all plugins

2. Click settings your discord - Plugins - Click "Open Plugins Folder"

3. Paste plugins

4. Turn on plugins
